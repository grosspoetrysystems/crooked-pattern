## Unreleased

- Pre-release quality pass (honesty + correctness): an unreachable or implausible `--url` is now an operational error instead of a fabricated 36/100 artifact; SPA catch-all rewrites no longer fake robots/sitemap/OpenAPI/OAuth/AGENTS.md/server-card passes (content plausibility is validated); the indirect-injection check quotes its evidence and no longer fails on ordinary CSS (`display:none`, sub-pixel `font-size`) or AI-adjacent prose; fixed `--rendered` crashing on every run (page-function serialization) and the CLS probe fabricating a high-confidence 0; cooldown detection now parses quoted JSON keys and day-unit values (previously NaN); provenance reports `unknown` when no CI workflows exist; authored-tool detection no longer counts UI naming conventions (phantom T4 passes / T5 vetoes); symlink cycles and dangling symlinks no longer break source scans; per-request fetch timeouts added.
